# CFA-CoderBnB

A clone of the popular accomodation booking site AirBnB.

## About the Project

A tutorial that supplements class learning about building an AirBnB clone.

## Requirements

- Ruby 2.0 or later.
- Rails 5.0 or later.
- Devise
- Bootstrap-sass
- Toastr-rails
- Sendgrid-rails
gem 'omniauth
gem 'omniauth-facebook'

## Installation

Add this line to your application's Gemfile:

  `gems to add`
